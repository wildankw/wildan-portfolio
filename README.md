# Wildan Khaustara W Portfolio

Static Next.js portfolio website with Indonesian and English language support.

## Run Locally

```bash
npm install
npm run dev
```

## Build Static Files

```bash
npm run build
```

The static website output will be generated in the `out` folder.

## Hostinger Upload

Upload the contents of the `out` folder into Hostinger `public_html`.
Make sure `index.html` is directly inside `public_html`, not inside another nested folder.
